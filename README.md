# Substack Prompts — Plutarch TX

Companion prompts for the [Plutarch TX](https://plutarchtx.substack.com) Substack.

Each article that ships with example prompts has a directory here named after the article slug (e.g. `021-vocabulary-youve-been-faking/`). Inside, you'll find the prompt files exactly as referenced inside the article.

## How this repo is updated

This repo is a **read-only public mirror**. It is populated automatically by the publishing pipeline at the moment each article goes live on Substack. Drafts and unpublished material live elsewhere and never reach this repo.

If a prompt directory is missing or a link from an article 404s briefly, the mirror is catching up — give it a minute.

## Using the prompts

- Each `prompt-N.md` is a complete prompt. Paste it into your model of choice (Claude, ChatGPT, etc.).
- The accompanying article provides the context and shows what the output should look like.
- Tweak freely. The point is the technique, not the exact wording.
